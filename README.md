# Turorial

To work on your own branch

1. clone the repo
```
git clone https://github.com/yousef184/motion_control.git
```

2. switch to your own branch
```
git switch <branch-name>
```



To push you work into the branch

Add all modified and new files:
```
git add .
```

Save your work with a clear, descriptive message: 
```
git commit -m "<a messege explaining what you did>"
```

Upload your commit to GitHub:
```
git push
```
